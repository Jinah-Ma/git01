1. 기본형(원시형) : 한 개
    • number : 숫자
    • string : 글자
    • bloolean : 참(true), 거짓(false)
  2. 복합형(참조형) : 여러 개
    • object : 객체(이름과 값으로 구성된 프로퍼티들의 집합)
    • array : 순서가 있는 요소들의 집합(접근시 []를 사용하여 인덱스 번호로 접근)
    • function : 함수(실행가능한 코드 블록, 기능을 반복해 사용 가능)
  3. 특수형
    • undefined : 변수가 선언되었으나 초기화 되지 않았을 때(변수에 값을 할당x)
    • null : 값이 없음.(변수에 명시적으로 할당해 사용)

typeof : 자료형을 확인하는 단항연산자
