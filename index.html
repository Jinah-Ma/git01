git

1. 폴더에서 최초한번 : git init
깃 시스템설치하는 것

★2. 수정한 파일의 스크린 샷 생성 : git add .
스크린샷을 버전으로 생성/ 계속 수정가능

★3. git commit -m "커밋메시지"
수정 끝 최종단계

4. 원격 저장소 생성 (폴더별 최초 한번)
깃허브로 이동하여 new repository클릭

5. 로컬 저장소와 원격저장소 연결 :
pc별 최초 한 번 사용자 등록 과정이 필요함.
유저의 이메일과 유저의 네임(내 것 복사해서 붙이기_오타 방지를 위해)
git remote add orgin 원격저장주소 (폴더별 1회 실행)

★6. 원격저장소로 코드 전송
git branch -M main (폴더별 1회 실행)
git push origin 브랜치명(지금은 main)

★붙여 놓은 것만 반복적으로 하면 됨
